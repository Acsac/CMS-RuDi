[=] Исправления мелких багов в том числе баг с повторной вставкой капчи в ajax
    формы;
[=] В плагине p_recaptcha удалена reCAPTCHA PHP library, проверка правильности 
    введенного пользователем кода осуществляется прямо в плагине, сам запрос
    осуществляется с помошью класса miniCurl (curl.class.php);
[+] Дефолтовый Smarty шаблон вынесен в отдельный шаблон _default_smarty_;
[=] Исправлена ошибка в компоненте Sitemap;
[=] Исправлена ошибка в файле sitemap компонента forum (для html карты);
[=] Добавлена установка компонента Sitemap при миграции, а также данные компонента
    Sitemap внесены в sql дампы установки;
[=] Исправлена ошибка с отсутствием пагинации на Smarty шаблонах;
[-] Из настроек системы удалена опция "Смещение в часах". Так же удален файл 
    /admin/includes/timezones.php теперь список временных зон генерируется
    с помошью функции DateTimeZone::listIdentifiers();
[+] Добавлена возможность пользователям выбирать свою временную зону. Так же 
    изменен метод ядра dateFormat чтобы переводить дату из системной временной зоны
    в временную зону пользователя;
[=] Часовые пояса переведены на русский язык с помошью переводчика Google. Возможны
    неточности в переводе, обо всех неточностях можете сообщить в тикетах;
[+] В класс rudi_form_generate добавлена возможность указания префикса для name
    атрибута полей и опция определяющая нужно ли вставлять в форму скрытое поле
    с csrf_token или нет;
[+] Для классов sitemap компонентов включена возможность иметь собственные настройки.
    Для примера в каталоге /components/sitemap/other_components/ лежит файл класса
    для компонента Instant Video video.sitemap.php (если у вас установлен этот
    компонент то переместите этот файл в папку /components/video/ и переименуйте
    его в sitemap.php);
